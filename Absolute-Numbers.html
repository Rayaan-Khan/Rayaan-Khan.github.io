ABSOLUTE NUMBERS


Absolute value describes the distance of a number on the number line from 0 without considering which direction from zero the number lies. The absolute value of a number is never negative. the absolute value of -7 is 7. Let's use a number line to picture that.

If you look at -7 on the number line and count how many numbers it’s from 0, it is 7 units (numbers) this means the absolute value is 7

Questions commonly appear in the following format:

  1. What is the absolute value of 8
  2. What is |8|

They seem different but just remember that the second one means the same as the first question and the answer is 8 for both as the number 8 is 8 units from 0.

To show that we want the absolute value of something, we put "|" marks either side (they are called "bars" and are found on the right side of a keyboard), like these examples:

|−2| = 2

|-156| = 156

Sometimes absolute value is also written as "abs()", so abs(0) = 0 is the same as |0| = 0

Multiplying, Addition, & Subtraction

  • |-3x6| = 18 Because -3 x 6 = -18 and |-18| = 18 the absolute value is 18
  • -|5-1| = -4 Because 5-2 = 3 and then the first minus gets us -4
  • -|5-2|=-4 Because 2-5 = -3, and then the first minus gets us -4

In The following scenarios although the absolute value is positive we are solve what is inside the bars first than add the outside so in this case because there is a negative the absolute value becomes negative.
